# Capstone

## Downloading the DEAM Dataset

This project uses the [DEAM (Database for Emotional Analysis in Music)](https://cvml.unige.ch/databases/DEAM/) dataset.

### Steps to Download

1. Visit the [DEAM dataset page](https://cvml.unige.ch/databases/DEAM/).
2. Scroll down to the "Download" section.
3. Fill out the request form with your name, email, and intended use.
4. After submitting the form, you will receive a download link via email.
5. Download the dataset (typically a ZIP file) from the link provided.
6. Extract the contents of the ZIP file.
7. Place the extracted audio files (e.g., `WaveFiles` folder) into the following directory in your project:
    ```
    data/WaveFiles/
    ```
   Create the directory if it does not exist.

### Notes

- The DEAM dataset is for research purposes only. Please respect the terms and conditions specified by the dataset authors.
- For more information about the dataset, refer to the [DEAM website](https://cvml.unige.ch/databases/DEAM/).
